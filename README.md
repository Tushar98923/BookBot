BookBot is a simple Python command-line tool that reads the contents of any text file and calculates the total number of characters within it. 

This project was built as a learning exercise for the backend development curriculum on [Boot.dev](https://boot.dev).

## Prerequisites

Before running this project, ensure you have Python installed on your machine. You will need **Python 3** to run this script.

## Usage

To use BookBot, open your terminal, navigate to the root directory of the project (where `main.py` is located), and run the following command:
```bash
python3 main.py <file_path>
